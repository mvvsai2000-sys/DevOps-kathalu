## Anaganaga...

Mana hero app deploy chestunte sudden ga error vachindi... 😳

⚠️ `FATAL: Port 8080 already in use`

“8080 port ni evaru use chestunnaru?” 🤯

Server lo chala applications run avuthunnayi...

Correct ga ela identify cheyyalo ardham kaaledu. 😵

Appude ataniki mobile battery settings gurthochindi. 💡

“Phone lo battery ekkuva use chestunna app ni ela identify chesthamo...

Alage Linux lo kuda port use chestunna process ni check cheyyachu!” 😎🔥

Ventane command run chesadu...

```bash id="bzjnrp"
lsof -i :8080
```

Boom! ⚡

```bash id="9fifkr"
java   8092   LISTEN
```

“Ahaaa!” 🤯

“Java application 8080 port ni occupy chesindi!” 🔥

Senior cheppadu...

“8092 ante PID... Ante aa process unique ID ra!” 😎

Ventane mana hero process stop chesadu...

```bash id="dbexmz"
kill -9 8092
```

2 seconds later... 👀

✅ APPLICATION ONLINE

“Ahaaaa! Finally app up ayyindi!” 🤯🔥

━━━━━━━━━━━━━━━

🔍 `lsof -i :8080`  
→ Port ni evaru use chestunnaro choodataniki

→ Process ni stop cheyyadaniki
